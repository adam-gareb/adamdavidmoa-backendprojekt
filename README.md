# adamdavidmoa-backendprojekt

This project simulates a library using a hsql database. It is not simulating a full scale library.

To run the project on unix based systems do this:
`chmod +x run.sh` or `chmod +x run_and_remove_database.sh` then `./run.sh` or `./run_and_remove_database.sh`

To run the project on windows based systems do this:

either open either of the .bat files or run this in cmd `run.bat` or `run_and_remove_database.bat`
